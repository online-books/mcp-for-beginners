## -0- 가상 환경 만들기

```bash
python -m venv venv
```

## -1- 가상 환경 활성화하기

```bash
venv\Scripts\activate
```

## -2- 의존성 설치하기

```bash
pip install "mcp[cli]"
```

## -3- 샘플 실행하기


```bash
mcp run server.py
```
