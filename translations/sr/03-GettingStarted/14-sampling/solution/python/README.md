# Покретање овог примера

Препоручује се инсталација `uv`, али није обавезно, види [упутства](https://docs.astral.sh/uv/#highlights)

## -0- Креирање виртуелног окружења

```bash
python -m venv venv
```

## -1- Активирање виртуелног окружења

```bash
venv\Scripts\activate
```

## -2- Инсталирање зависности

```bash
pip install "mcp[cli]"
```

## -3- Покретање примера


```bash
mcp run server.py
```
