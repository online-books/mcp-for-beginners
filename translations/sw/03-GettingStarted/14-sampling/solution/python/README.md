# Kuendesha sampuli hii

Unaashauri kufunga `uv` lakini si lazima, angalia [maelekezo](https://docs.astral.sh/uv/#highlights)

## -0- Unda mazingira ya mtandao

```bash
python -m venv venv
```

## -1- Washa mazingira ya mtandao

```bash
venv\Scripts\activate
```

## -2- Sakinisha utegemezi

```bash
pip install "mcp[cli]"
```

## -3- Endesha sampuli


```bash
mcp run server.py
```

## -4- Jaribu sampuli

Iongeze kwenye *mcp.json* kama ifuatavyo:

```json
```
