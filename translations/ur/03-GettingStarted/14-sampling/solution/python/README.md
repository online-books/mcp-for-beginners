# اس سیمپل کو چلانا

آپ کو `uv` انسٹال کرنے کی سفارش کی جاتی ہے لیکن یہ ضروری نہیں ہے، ہدایات دیکھیں [instructions](https://docs.astral.sh/uv/#highlights)

## -0- ایک ورچوئل ماحول تخلیق کریں

```bash
python -m venv venv
```

## -1- ورچوئل ماحول کو فعال کریں

```bash
venv\Scripts\activate
```

## -2- انحصاریات انسٹال کریں

```bash
pip install "mcp[cli]"
```

## -3- سیمپل چلائیں

```bash
mcp run server.py
```

## -4- سیمپل کا ٹیسٹ کریں
