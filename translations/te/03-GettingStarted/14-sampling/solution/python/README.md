# ఈ నమూనాను నడపడం

మీరు `uv` ని ఇన్ స్టాల్ చేయాలని సలహా ఇస్తున్నారు, కానీ అది తప్పనిసరి కాదు, [సూచనలు](https://docs.astral.sh/uv/#highlights)ను చూడండి

## -0- వర్చువల్ ఎన్విరాన్‌మెంట్ సృష్టించండి

```bash
python -m venv venv
```

```bash
venv\Scripts\activate
```

```bash

```

```bash
mcp run server.py
```
