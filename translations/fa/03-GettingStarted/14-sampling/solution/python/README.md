# اجرای این نمونه

پیشنهاد می‌شود `uv` را نصب کنید اما اجباری نیست، به [دستورالعمل‌ها](https://docs.astral.sh/uv/#highlights) مراجعه کنید

## -0- ایجاد یک محیط مجازی

```bash
python -m venv venv
```

## -1- فعال کردن محیط مجازی

```bash
venv\Scripts\activate
```

## -2- نصب وابستگی‌ها

```bash
pip install "mcp[cli]"
```

## -3- اجرای نمونه


```bash
mcp run server.py
```
