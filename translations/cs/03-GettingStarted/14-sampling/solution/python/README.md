# Spuštění tohoto příkladu

Doporučuje se nainstalovat `uv`, ale není to nutné, viz [návod](https://docs.astral.sh/uv/#highlights)

## -0- Vytvořte virtuální prostředí

```bash
python -m venv venv
```

## -1- Aktivujte virtuální prostředí

```bash
venv\Scripts\activate
```

## -2- Nainstalujte závislosti

```bash
pip install "mcp[cli]"
```

## -3- Spusťte příklad


```bash
mcp run server.py
```

## -4- Otestujte příklad

Spusťte server příkazem:

Přidejte to do *mcp.json* takto:

```json
```

Spusťte server

