# यस नमूनालाई चलाउँदै

तपाईंलाई `uv` इन्स्टल गर्न सिफारिस गरिन्छ तर यो आवश्यक छैन, हेर्नुहोस् [निर्देशहरू](https://docs.astral.sh/uv/#highlights)

## -0- एउटा भर्चुअल वातावरण सिर्जना गर्नुहोस्

```bash
python -m venv venv
```

## -1- भर्चुअल वातावरण सक्रिय गर्नुहोस्

```bash
venv\Scripts\activate
```

## -2- निर्भरता इन्स्टल गर्नुहोस्

```bash
pip install "mcp[cli]"
```

## -3- नमूना चलाउनुहोस्


```bash
mcp run server.py
```
