## -0- एक आभासी पर्यावरण तयार करा

```bash
python -m venv venv
```

## -1- आभासी पर्यावरण सक्रिय करा

```bash
venv\Scripts\activate
```

## -2- अवलंबित पॅकेजेस इन्स्टॉल करा

```bash
pip install "mcp[cli]"
```

## -3- नमुना चालवा


```bash
mcp run server.py
```
