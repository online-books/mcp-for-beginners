## -0- Sanal ortam oluşturun

```bash
python -m venv venv
```

## -1- Sanal ortamı etkinleştirin

```bash
venv\Scripts\activate
```

## -2- Bağımlılıkları kurun

```bash
pip install "mcp[cli]"
```

## -3- Örneği çalıştırın


```bash
mcp run server.py
```
