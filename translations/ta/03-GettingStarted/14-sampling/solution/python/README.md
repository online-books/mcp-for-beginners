# இந்த மாதிரியை இயக்கு

நீங்கள் `uv` ஐ நிறுவ பரிந்துரைக்கப்படுகிறீர்கள் ஆனாலும் அது அவசியம் இல்லை, [வழிமுறைகள்](https://docs.astral.sh/uv/#highlights) பார்க்கவும்

## -0- ஒரு மெய்நிகர் சூழலை உருவாக்கவும்

```bash
python -m venv venv
```

## -1- மெய்நிகர் சூழலை செயல்படுத்தவும்

```bash
venv\Scripts\activate
```

## -2- சார்புகளை நிறுவவும்

```bash
pip install "mcp[cli]"
```

## -3- மாதிரியை இயக்கவும்


```bash
mcp run server.py
```

## -4- மாதிரியை சோதிக்கவும்

சர்வரை கீழ்க்கண்ட கட்டளையுடன் இயக்கவும்:

*mcp.json* இல் இதை பின்வருமாறு சேர்க்கவும்:

```json
```

```text
prompt
```
