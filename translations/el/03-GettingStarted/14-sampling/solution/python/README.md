# Εκτέλεση αυτού του δείγματος

Συνιστάται να εγκαταστήσετε το `uv` αλλά δεν είναι απαραίτητο, δείτε τις [οδηγίες](https://docs.astral.sh/uv/#highlights)

## -0- Δημιουργήστε ένα εικονικό περιβάλλον

```bash
python -m venv venv
```

## -1- Ενεργοποιήστε το εικονικό περιβάλλον

```bash
venv\Scripts\activate
```

## -2- Εγκαταστήστε τις εξαρτήσεις

```bash
pip install "mcp[cli]"
```

## -3- Εκτελέστε το δείγμα


```bash
mcp run server.py
```

## -4- Δοκιμάστε το δείγμα

Εκτελέστε τον διακομιστή με την εντολή:

Προσθέστε το στο *mcp.json* ως εξής:

```json
```
