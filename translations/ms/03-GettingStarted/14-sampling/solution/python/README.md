```bash
python -m venv venv
```

## -1- Aktifkan persekitaran maya

```bash
venv\Scripts\activate
```

## -2- Pasang kebergantungan

```bash
pip install "mcp[cli]"
```

## -3- Jalankan contoh


```bash
mcp run server.py
```
