# Köra detta exempel

Det rekommenderas att du installerar `uv` men det är inte ett måste, se [anvisningar](https://docs.astral.sh/uv/#highlights)

## -0- Skapa en virtuell miljö

```bash
python -m venv venv
```

## -1- Aktivera den virtuella miljön

```bash
venv\Scripts\activate
```

## -2- Installera beroenden

```bash
pip install "mcp[cli]"
```

## -3- Kör exemplet


```bash
mcp run server.py
```
