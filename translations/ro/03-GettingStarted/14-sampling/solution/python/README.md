# Rularea acestui exemplu

Se recomandă instalarea `uv`, dar nu este obligatoriu, vezi [instrucțiuni](https://docs.astral.sh/uv/#highlights)

## -0- Creează un mediu virtual

```bash
python -m venv venv
```

## -1- Activează mediul virtual

```bash
venv\Scripts\activate
```

## -2- Instalează dependențele

```bash
pip install "mcp[cli]"
```

## -3- Rulează exemplul


```bash
mcp run server.py
```

## -4- Testează exemplul

Rulează serverul cu comanda:

Adaugă-l în *mcp.json* astfel:

```json
```

Pornește serverul
