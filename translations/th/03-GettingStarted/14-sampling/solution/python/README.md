# การรันตัวอย่างนี้

คุณแนะนำให้ติดตั้ง `uv` แต่ไม่จำเป็น ดูที่ [คำแนะนำ](https://docs.astral.sh/uv/#highlights)

## -0- สร้างสภาพแวดล้อมเสมือน

```bash
python -m venv venv
```

## -1- เปิดใช้งานสภาพแวดล้อมเสมือน

```bash
venv\Scripts\activate
```

## -2- ติดตั้ง dependencies

```bash
pip install "mcp[cli]"
```

## -3- รันตัวอย่าง


```bash
mcp run server.py
```
